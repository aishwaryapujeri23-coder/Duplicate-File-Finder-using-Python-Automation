Automated Duplicate File Finder:
- An intelligent desktop application that scans your system to detect and manage duplicate files efficiently. Built with Python and PyQt6, this tool helps users free up storage space and organize files with ease.
---
Features:
- Fast Duplicate Detection (SHA-256 hashing)
- Multithreaded Scanning
- Modern GUI (PyQt6)
- Progress Tracking
- File Selection with checkboxes
- Chatbot
---
Technologies Used:
- Python3
- PyQt6
- Hashlib
- Multithreading
  
---
Project Structure:
Automated_Duplicate File Finder/
- app.py
- chatbot.py
- venv/
- pycache/
  
---

Installation:
1. Clone the repository:
git clone https://github.com/aishwaryapujeri23-coder/Duplicate-File-Finder-using-Python-Automation.git
cd duplicate-file-finder
2. Create virtual environment:
python -m venv venv
3. Activate:
Windows: venv\Scripts\activate  
Linux/Mac: source venv/bin/activate
4. Install dependencies:
pip install 

---
Run:
python app.py

---
How It Works:
- Select folder
- Scan files
- Generate SHA-256 hashes
- Identify duplicates
- Display results
